2.12 AI-Driven Hierarchical Ensemble Framework for ICS Protocol Anomaly Detection VCR

SSL-TRIDENT is a semi/self-supervised hierarchical ensemble framework for ICS protocol anomaly detection on MODBUS, DNP3, and IEC 60870-5-104. It replaces label-intensive supervised models with three optimized components: a self-supervised contrastive encoder, a positive-unlabeled classifier (requires only 10-100 attack labels), and a lightweight signature matcher. Operating in inline (<10ms latency, int8 quantization) or passive mode, it achieves F1 scores of 0.94-0.96 with false-positive rates ≤0.010 and zero-day recall ≥70% on CIC Modbus 2023—using 99.8% fewer labels than fully supervised baselines.

AI & Machine Learning | ICS/OT Security

Key Components: Self-supervised contrastive encoder (Level 1), Positive-unlabeled binary classifier (Level 2, 10-100 attacks), Anomaly signature matcher (Level 3), Probabilistic four-case fusion matrix, Online PU updates, MMD drift detection.

Technologies: PyTorch MODBUS/DNP3/IEC CIC Modbus 2023 int8 Quantization Real-Time Linux
